# library Management 도서관리

새로 들어온 도서 등록과 도서목록 조회
<br></br>

## 팀원 소개
이해민, 이주용, 이은지 : FE, BE

## Stacks

IDE: Eclipse(2021-03)<br></br>
DB: MySQL(v8.0.30)<br></br>
MavenProject, Spring(v5.3.14), CSS, JPA<br></br>
